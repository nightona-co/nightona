The Nightona Go SDK provides a powerful interface for programmatically interacting with Nightona Sandboxes. It requires Go 1.25 or later.

## Installation

Install the Nightona Go SDK using go get:

```bash
go get github.com/nightona-co/nightona/libs/sdk-go
```

## Getting Started

### Create a Sandbox

Create a Nightona Sandbox to run your code securely in an isolated environment. The following snippet is an example "Hello World" program that runs securely inside a Nightona Sandbox.

```go
package main

import (
	"context"
	"fmt"
	"log"

	"github.com/nightona-co/nightona/libs/sdk-go/pkg/nightona"
)

func main() {
	// Initialize the SDK (uses environment variables by default)
	client, err := nightona.NewClient()
	if err != nil {
		log.Fatal(err)
	}

	// Create a new sandbox
	sandbox, err := client.Create(context.Background(), nil)
	if err != nil {
		log.Fatal(err)
	}

	// Execute a command
	response, err := sandbox.Process.ExecuteCommand(context.Background(), "echo 'Hello, World!'")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(response.Result)
}
```

## Configuration

The Nightona SDK can be configured using environment variables or by passing options to the constructor:

```go
package main

import (
	"github.com/nightona-co/nightona/libs/sdk-go/pkg/nightona"
	"github.com/nightona-co/nightona/libs/sdk-go/pkg/types"
)

func main() {
	// Using environment variables (NIGHTONA_API_KEY, NIGHTONA_API_URL, NIGHTONA_TARGET)
	client, _ := nightona.NewClient()

	// Using explicit configuration
	config := &types.NightonaConfig{
		APIKey: "YOUR_API_KEY",
		APIUrl: "https://app.daytona.io/api",
		Target: "us",
	}
	client, _ = nightona.NewClientWithConfig(config)
}
```

For more information on configuring the Nightona SDK, see [configuration](/docs/en/configuration).
